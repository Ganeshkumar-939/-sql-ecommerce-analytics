# E-Commerce Sales & Customer Analytics (SQL)

## Objective
Analyze e-commerce sales and customer behavior using SQL to track revenue performance, identify high-value customers, and detect churn risk.

## Dataset
- customers
- orders
- order_items
- products

## Key KPIs
- Total Revenue
- Average Order Value (AOV)
- Repeat Purchase Rate
- Customer Churn Risk

## Tools Used
- SQL (MySQL / PostgreSQL)

## Repository Structure
- data/ – CSV files
- sql/ – SQL queries
- screenshots/ – Query outputs
